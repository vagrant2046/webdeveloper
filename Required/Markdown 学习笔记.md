# 	Markdown 学习笔记

## 标题

```mark
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 段落格式

### *段落*

>   段落的换行是使用两个以上空格加上回车

### *字体*

*斜体*     **粗体**    ***粗斜体***

```
*斜体文本*
**粗体文本**
***粗斜体文本***
```

### *分割线*

****

```
*** 下划线
```

### *删除线*

~~删除线~~

```
~~删除线~~
```

###   *下划线*

<u>下划线</u>

```
<u>带下划线文本</u>
```

### *脚注*

```
[^要注明的文本]
```
